# **Environment MacOS**

## 1. Desabilitar seguridad del sistema

- Reiniciar en Modo Recovery. Apagar la macbook y volverla a encender, y ni bien enciende, presionar ` command + R`
- Abrir la terminal
- Correr el comando `csrutil disable`
- Reiniciar  
  
       
> Los siguientes pasos se hacen en la terminal

## 2. Instalar Homebrew
Ejecutar el siguiente comando:

`
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
`
